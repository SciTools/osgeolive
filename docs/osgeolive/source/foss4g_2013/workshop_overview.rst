===================================
FOSS4G 2013: Iris Big Data Workshop
===================================

In this workshop we will introduce Iris, along with interesting multi-dimensional datasets, and show how "big data" can be served up using your existing open source software stack using GDAL, QGIS, GeoServer, OpenLayers and ZOO-Project.
